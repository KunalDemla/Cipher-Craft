FAQs

What is Cipher Craft?

Cipher Craft is a Django-based web application written on Python 3. Visitors can make use of its features to encrypt English to various ciphers and vice-versa.

Is my data shared with Cipher Craft Securly?

Yes, all the data you share with us is kept secured and all the passwords are stored in hashed format. No personal activity is tracked or stored.

Can I contribute to the project?

You most definitely can. Please visit the project's repository on GitHub for more details. You can learn more about its contributing guidelines there.
